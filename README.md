# Chat App sử dụng React Native

Ứng dụng chat đơn giản được xây dựng bằng React Native.

## Giới thiệu

Ứng dụng này là một ví dụ cơ bản về ứng dụng chat được phát triển bằng framework React Native. Nó cho phép người dùng gửi và nhận tin nhắn trong thời gian thực.

## Công nghệ sử dụng

* **React Native:** Framework JavaScript để xây dựng ứng dụng di động đa nền tảng.
* **[Tên thư viện quản lý state (ví dụ: Redux, Zustand, Context API)] (Tùy chọn):** Để quản lý trạng thái của ứng dụng.
* **[Tên thư viện UI (ví dụ: NativeBase, React Native Elements, hoặc React Native Paper)] (Tùy chọn):** Để xây dựng giao diện người dùng.
* **[Tên backend/realtime database (ví dụ: Firebase Realtime Database, Socket.IO, Pusher)] (Tùy chọn):** Để xử lý việc gửi và nhận tin nhắn trong thời gian thực.

## Tính năng

* Gửi và nhận tin nhắn văn bản.
* Hiển thị danh sách tin nhắn.
* Giao diện người dùng đơn giản và dễ sử dụng.

## Cài đặt

1.  **Yêu cầu:** Đảm bảo bạn đã cài đặt Node.js, npm (hoặc yarn), và môi trường phát triển React Native trên máy tính của mình. Tham khảo hướng dẫn cài đặt React Native tại [https://reactnative.dev/docs/environment-setup](https://reactnative.dev/docs/environment-setup).

2.  **Clone repository:**
    ```bash
    git clone [Địa chỉ repository của bạn]
    cd [Tên thư mục repository]
    ```

3.  **Cài đặt các dependencies:**
    ```bash
    npm install
    # hoặc
    yarn install
    ```

4.  **Chạy ứng dụng trên thiết bị iOS:**
    ```bash
    npx react-native run-ios
    # hoặc
    yarn ios
    ```

5.  **Chạy ứng dụng trên thiết bị Android:**
    ```bash
    npx react-native run-android
    # hoặc
    yarn android
    ```
